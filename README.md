# Legal Assistant Prototype

Этот репозиторий содержит упрощённый пример сервиса для работы с документами. 
Он демонстрирует возможный подход к созданию ассистента для юристов, 
который умеет загружать файлы, извлекать из них текст и организовывать их 
по делам клиентов.

## Основные возможности

- Загрузка PDF или изображений с сохранением по имени клиента (ФИО).
- Извлечение текста из документов при помощи `PyPDF2` и `pytesseract`.
- Хранение списка дел и документов в базе SQLite.
- Просмотр списка документов и их текста через CLI.
- Пример шаблона документа в каталоге `templates`.

## Быстрый старт

1. Установите зависимости:
   ```bash
   pip install PyPDF2 pytesseract Pillow click Jinja2
   ```
2. Запустите загрузку файла:
   ```bash
   python main.py upload path/to/file.pdf --client "Иванов Иван"
   ```
3. Посмотрите список документов:
   ```bash
   python main.py list --client "Иванов Иван"
   ```
4. Отобразите текст документа по его идентификатору:
   ```bash
   python main.py show 1
   ```

Этот код служит отправной точкой. Его можно доработать, добавив 
веб-интерфейс, панель администратора и генерацию документов по шаблонам.
